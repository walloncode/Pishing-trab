Sobre o projeto

Este projeto foi desenvolvido como atividade acadêmica para a disciplina de Segurança da Informação no Instituto Federal de Rondônia.

O objetivo da simulação foi demonstrar, de forma prática, como ataques de phishing podem utilizar engenharia social para induzir usuários a fornecer informações em páginas aparentemente legítimas.

A campanha simulava um suposto acesso gratuito ao Wi-Fi Starlink institucional através de uma página de login semelhante a portais reais de autenticação.

Objetivo

O projeto teve como finalidade:

Demonstrar técnicas comuns utilizadas em campanhas de phishing;
Avaliar o nível de atenção dos usuários diante de páginas falsas;
Promover conscientização sobre segurança digital;
Incentivar boas práticas de verificação de links e autenticação.
Funcionamento da simulação

A aplicação apresentava:

Página de login temática;
Interface semelhante a um portal de autenticação Wi-Fi;
Coleta apenas de informações não sensíveis para fins estatísticos;
Redirecionamento posterior para uma tela explicativa sobre a simulação.

Nenhuma credencial real deveria ser reutilizada durante os testes.

Contexto acadêmico

Este projeto foi desenvolvido exclusivamente para fins educacionais e de conscientização em segurança da informação.

A proposta não possui finalidade maliciosa, comercial ou de coleta indevida de dados.

Aviso ético

Este repositório existe apenas como demonstração acadêmica.

O uso de técnicas de phishing fora de ambientes autorizados, controlados e educativos pode violar leis, políticas institucionais e princípios éticos de segurança da informação.
